# 💰 Intelligent Expense Tracker (Data Science Project)

## 📌 Overview

This project is a **data-driven expense tracking system** that helps users analyze their spending behavior using **Python, Data Science, and Machine Learning**.

---

## 🚀 Features

* Synthetic financial data generation
* Interactive dashboard using Streamlit
* Category-wise spending analysis
* Monthly & daily trend analysis
* Budget vs Actual comparison
* Smart alerts for overspending
* Financial health score
* **Anomaly Detection using Isolation Forest (ML)**

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Streamlit
* Plotly
* Scikit-learn

---

## 📊 Key Insights

* Identifies highest spending category
* Detects unusual transactions
* Tracks spending trends
* Provides financial health score

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run main.py
```
